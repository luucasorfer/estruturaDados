Introdução à Linguagem C

A linguagem C é uma linguagem de programação de propósito geral, estruturada e imperativa. Desenvolvida em 1972 por Dennis Ritchie nos laboratórios da AT&T Bell, a linguagem C se tornou uma das linguagens mais influentes na história da computação.

C é considerada uma linguagem de "médio nível", combinando elementos de linguagens de alto nível com a funcionalidade de linguagem de montagem.

Primeiro Programa em C

Todo programa em C começa com a função main(). Aqui está um exemplo simples do tradicional "Hello World":

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Vantagens

  • Alto desempenho
  • Controle de baixo nível
  • Portabilidade
  • Grande base de código existente

Desafios

  • Gerenciamento manual de memória
  • Ausência de orientação a objetos
  • Verificações em tempo de execução limitadas
  • Curva de aprendizado mais íngreme

Conceitos Básicos

Variáveis

Variáveis são nomes simbólicos que representam locais na memória onde armazenamos dados. Em C, todas as variáveis devem ser declaradas antes de serem usadas.

// Declaração de variáveis
int idade;
float altura;
char letra;

// Inicialização
int contador = 0;
double pi = 3.14159;
char inicial = 'A';

Em C, os nomes de variáveis são case-sensitive (idade é diferente de Idade) e devem começar com uma letra ou sublinhado (_).

Tipos de Dados

C possui vários tipos de dados básicos que podem ser classificados em:

Tipo Descrição Tamanho (bytes) Exemplo
char Caractere único 1 'A', '1', '$'
int Número inteiro 2 ou 4 10, -5, 0
float Número de ponto flutuante 4 3.14, -0.001
double Ponto flutuante de dupla precisão 8 3.1415926535
void Sem valor/tipo - Usado em funções

O tamanho exato dos tipos pode variar dependendo da plataforma e do compilador. Use sizeof() para verificar o tamanho em seu sistema.

Operadores

C possui diversos operadores para realizar operações em variáveis e valores:

Aritméticos

  • + (adição)
  • - (subtração)
  • * (multiplicação)
  • / (divisão)
  • % (módulo)
  • ++ (incremento)
  • -- (decremento)

Relacionais

  • == (igual a)
  • != (diferente de)
  • > (maior que)
  • < (menor que)
  • >= (maior ou igual)
  • <= (menor ou igual)

Lógicos

  • && (E lógico)
  • || (OU lógico)
  • ! (NÃO lógico)
// Exemplo de uso de operadores
int a = 10, b = 20, c;
float resultado;

c = a + b;          // Operador aritmético
if (a > b) {      // Operador relacional
    // ...
}
if (a > 0 && b > 0) {  // Operadores lógicos
    // ...
}

Controle de Fluxo

Condicionais

As estruturas condicionais permitem que o programa tome decisões com base em condições.

if-else

if (condicao) {
    // código executado se condição for verdadeira
} else if (outra_condicao) {
    // código executado se outra_condicao for verdadeira
} else {
    // código executado se nenhuma das condições for verdadeira
}

switch-case

switch (expressao) {
    case valor1:
        // código para valor1
        break;
    case valor2:
        // código para valor2
        break;
    default:
        // código se nenhum case corresponder
}

O break no switch é crucial para evitar a execução em cascata dos casos subsequentes. Se omitido, a execução continuará para o próximo case.

Loops

Loops permitem a execução repetida de um bloco de código enquanto uma condição for verdadeira.

while

while (condicao) {
    // código a ser repetido
}

do-while

do {
    // código a ser repetido
} while (condicao);

for

for (inicializacao; condicao; incremento) {
    // código a ser repetido
}

Loop Control

  • break - Sai imediatamente do loop
  • continue - Pula para a próxima iteração
  • goto - (Evitar quando possível) Pula para um rótulo

Exemplo Prático

// Imprime números pares de 0 a 10
for (int i = 0; i <= 10; i++) {
    if (i % 2 != 0) {
        continue;  // Pula números ímpares
    }
    printf("%d ", i);
}

Funções

Funções são blocos de código que realizam uma tarefa específica e podem ser chamados de diferentes partes do programa. Elas promovem a reutilização de código e a modularização.

// Declaração (protótipo)
int somar(int a, int b);

// Definição
int somar(int a, int b) {
    return a + b;
}

// Chamada
int resultado = somar(5, 3);

Parâmetros e Retorno

  • Funções podem receber zero ou mais parâmetros
  • Podem retornar um valor ou ser do tipo void
  • Parâmetros podem ser passados por valor ou por referência (usando ponteiros)
  • O tipo de retorno deve corresponder ao declarado

Recursão

Funções podem chamar a si mesmas (recursão):

int fatorial(int n) {
    if (n <= 1) return 1;
    return n * fatorial(n - 1);
}

Boas Práticas com Funções

  • Dê nomes descritivos às funções
  • Mantenha as funções pequenas e com um único propósito
  • Documente os parâmetros e o valor de retorno
  • Use protótipos para declarar funções antes de defini-las

Arrays

Arrays são estruturas que armazenam uma coleção de elementos do mesmo tipo em posições contíguas de memória.

// Declaração e inicialização
int numeros[5] = {10, 20, 30, 40, 50};

// Acesso aos elementos
int primeiro = numeros[0];  // Índices começam em 0
numeros[2] = 35;            // Modificando o terceiro elemento

// Array multidimensional (matriz)
int matriz[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Características

  • Tamanho fixo (definido na declaração)
  • Índices começam em 0
  • Elementos são armazenados em posições contíguas de memória
  • O nome do array é um ponteiro para seu primeiro elemento
  • Não há verificação de limites (cuidado com buffer overflow)

Strings

Em C, strings são arrays de caracteres terminados com '\0':

char nome[20] = "João Silva";
char *mensagem = "Olá, mundo!";

// Funções de string (necessário #include <string.h>)
strlen(nome);    // Comprimento
strcpy(dest, src); // Cópia
strcat(s1, s2);  // Concatenação
strcmp(s1, s2);  // Comparação

Ponteiros

Ponteiros são variáveis que armazenam endereços de memória. Eles são fundamentais em C para manipulação eficiente de memória e estruturas de dados.

int var = 10;
int *ptr = &var;   // ptr armazena o endereço de var

printf("Valor: %d\n", *ptr);  // Acessa o valor apontado
*ptr = 20;                   // Modifica o valor apontado

Operações com Ponteiros

  • & - Operador de endereço
  • * - Operador de dereferência
  • Aritmética de ponteiros (incremento, decremento)
  • Comparação entre ponteiros
  • Ponteiro para ponteiro (**)

Ponteiros e Arrays

int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr;  // Equivalente a &arr[0]

// Acesso alternativo
printf("%d\n", *(ptr + 2));  // arr[2]

// Ponteiros para funções
int (*func_ptr)(int, int) = &somar;

Cuidados com Ponteiros

  • Nunca dereferencie um ponteiro não inicializado
  • Evite ponteiros "soltos" (dangling pointers)
  • Verifique alocações de memória dinâmica
  • Tenha cuidado com vazamentos de memória

Estruturas

Estruturas (structs) permitem agrupar variáveis de diferentes tipos sob um único nome, criando tipos de dados complexos.

// Definição da estrutura
struct Pessoa {
    char nome[50];
    int idade;
    float altura;
};

// Declaração e uso
struct Pessoa p1;
strcpy(p1.nome, "Maria");
p1.idade = 25;
p1.altura = 1.68;

// Ponteiros para estruturas
struct Pessoa *ptr = &p1;
printf("Nome: %s\n", ptr->nome);

Typedef

O typedef permite criar aliases para tipos existentes, incluindo estruturas:

typedef struct {
    int dia, mes, ano;
} Data;

Data hoje = {15, 8, 2023};

Unions

Unions permitem que diferentes tipos de dados ocupem o mesmo espaço de memória:

union Valor {
    int i;
    float f;
    char str[20];
};

union Valor v;
v.i = 10;   // Agora armazena um int
v.f = 3.14; // Agora armazena um float, substituindo o int

Arquivos

C fornece funções para manipulação de arquivos através da biblioteca stdio.h. As operações básicas incluem abrir, ler, escrever e fechar arquivos.

FILE *arquivo;

// Abrir arquivo para escrita
arquivo = fopen("dados.txt", "w");
if (arquivo == NULL) {
    printf("Erro ao abrir arquivo!\n");
    return 1;
}

// Escrever no arquivo
fprintf(arquivo, "Olá, arquivo!\n");

// Fechar o arquivo
fclose(arquivo);

// Ler do arquivo
arquivo = fopen("dados.txt", "r");
char linha[100];
while (fgets(linha, sizeof(linha), arquivo) != NULL) {
    printf("%s", linha);
}
fclose(arquivo);

Modos de Abertura

  • "r" - Leitura (arquivo deve existir)
  • "w" - Escrita (cria ou sobrescreve)
  • "a" - Anexar (cria ou escreve no final)
  • "r+" - Leitura e escrita
  • "w+" - Leitura e escrita (cria ou sobrescreve)
  • "a+" - Leitura e anexação
  • "b" - Modo binário (ex: "rb", "wb")

Funções Principais

  • fopen() / fclose() - Abrir/fechar arquivo
  • fprintf() / fscanf() - Escrita/leitura formatada
  • fputs() / fgets() - Escrita/leitura de strings
  • fwrite() / fread() - Escrita/leitura binária
  • fseek() / ftell() - Navegação no arquivo
  • feof() - Verifica fim do arquivo

Boas Práticas

  • Sempre verifique se o arquivo foi aberto com sucesso
  • Feche os arquivos após o uso para liberar recursos
  • Em operações críticas, verifique erros após cada operação
  • Para arquivos binários, use fread() e fwrite()
  • Considere usar buffers para operações intensivas
