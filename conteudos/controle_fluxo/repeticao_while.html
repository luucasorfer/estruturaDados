Estrutura de Repetição: while

A estrutura de repetição while executa um bloco de código enquanto uma determinada condição for verdadeira. É usada quando não se sabe exatamente quantas vezes o laço deve ser executado, mas a repetição depende de uma condição lógica.

A sintaxe básica do while é a seguinte: 

// Sintaxe básica
while (condição) {
    // bloco de código a ser repetido enquanto a condição for verdadeira
}

Antes de cada iteração, a condição é avaliada. Se for verdadeira, o bloco de código dentro das chaves é executado. Caso contrário, o laço termina e a execução continua após o while.

Exemplo:

Abaixo, um exemplo que utiliza while para imprimir os números de 1 a 5: 

// Exibe números de 1 a 5
#include <stdio.h>

int main() {
    int i = 1;

    while (i <= 5) {
        printf("Número: %d\n", i);
        i++;
    }

    return 0;
}

Neste exemplo, a variável i é inicializada com 1. Enquanto i <= 5 for verdadeiro, o código dentro do while é executado, exibindo o valor de i e incrementando-o em 1. Quando i ultrapassa 5, a condição torna-se falsa e o laço é encerrado.
