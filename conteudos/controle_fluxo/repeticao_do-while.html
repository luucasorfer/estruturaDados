Estrutura de Repetição: do-while

A estrutura de repetição do-while executa um bloco de código pelo menos uma vez e repete enquanto uma condição for verdadeira. Diferentemente do while, a condição é verificada após a execução do bloco, garantindo que ele seja executado pelo menos uma vez.

A sintaxe básica do do-while é a seguinte: 

// Sintaxe básica
do {
    // bloco de código a ser executado
} while (condição);

O bloco dentro do do é executado primeiro e, em seguida, a condição é avaliada. Se for verdadeira, o bloco é executado novamente. Isso se repete até que a condição se torne falsa.

Exemplo:

Abaixo, um exemplo que utiliza do-while para imprimir os números de 1 a 5: 

// Exibe números de 1 a 5
#include <stdio.h>

int main() {
    int i = 1;

    do {
        printf("Número: %d\n", i);
        i++;
    } while (i <= 5);

    return 0;
}

Neste exemplo, o bloco dentro do do é executado, imprimindo o valor de i e incrementando-o. Após a execução, a condição i <= 5 é verificada; enquanto for verdadeira, o laço continua repetindo.
