Estrutura de Controle: if, else, else if

A estrutura de controle if em C é usada para executar um bloco de código condicionalmente, ou seja, apenas se uma determinada condição for verdadeira. Ela permite que o programa tome decisões e execute diferentes caminhos com base em condições específicas. A sintaxe básica do if é a seguinte: 

// Exemplo básico com if
if (condição) {
    // bloco de código a ser executado se a condição for verdadeira
}

A condição dentro dos parênteses é avaliada como verdadeira (true) ou falsa (false). Se a condição for verdadeira, o bloco de código dentro das chaves será executado. Caso contrário, o bloco será ignorado.

Além do if, é possível usar a estrutura else para definir um bloco de código que será executado quando a condição do if for falsa: 

// if com else
if (condição) {
    // bloco de código se a condição for verdadeira
} else {
    // bloco de código se a condição for falsa
}

Também é possível encadear múltiplas condições usando else if, permitindo diferentes caminhos lógicos com base em várias condições: 

// if, else if e else
if (condição1) {
    // executa se condição1 for verdadeira
} else if (condição2) {
    // executa se condição2 for verdadeira
} else {
    // executa se nenhuma das condições anteriores for verdadeira
}

A estrutura if, else if, else é uma ferramenta poderosa para controlar o fluxo de execução em programas C, permitindo que diferentes ações sejam tomadas com base em condições variadas.

Exemplo:

Abaixo, veja um exemplo prático completo, que solicita a nota do usuário e exibe uma avaliação baseada no valor informado. 

// Programa que avalia uma nota
#include <stdio.h>

int main() {
    int nota;

    printf("Digite sua nota: ");
    scanf("%d", &nota);

    if (nota >= 90) {
        printf("Excelente\n");
    } else if (nota >= 70) {
        printf("Bom\n");
    } else {
        printf("Precisa melhorar\n");
    }

    return 0;
}

Esse programa usa scanf para ler a nota digitada pelo usuário e, em seguida, avalia se ela é maior ou igual a 90, entre 70 e 89, ou menor que 70. O uso de if, else if e else garante que apenas um dos blocos será executado.

Operador Ternário

O operador ternário em C é uma forma concisa de escrever uma estrutura if-else simples. Ele é utilizado na forma: condição ? valor_se_verdadeiro : valor_se_falso.

Essa estrutura é ideal para atribuições ou saídas rápidas baseadas em uma condição simples. Veja um exemplo abaixo: 

// Exemplo com operador ternário
#include <stdio.h>

int main() {
    int idade;

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    // Uso do operador ternário para exibir a mensagem
    (idade >= 18) 
        ? printf("Você é maior de idade.\n") 
        : printf("Você é menor de idade.\n");

    return 0;
}

Neste exemplo, o programa utiliza o operador ternário para verificar se a idade informada é maior ou igual a 18 e exibe a mensagem apropriada em uma única linha de código. É uma alternativa elegante e compacta ao uso do if-else tradicional.
