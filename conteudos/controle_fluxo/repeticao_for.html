Estrutura de Repetição: for

A estrutura de repetição for é usada para executar um bloco de código um número conhecido de vezes. É ideal quando se sabe de antemão quantas vezes uma ação deve ser repetida, como ao percorrer um intervalo de valores.

Sua sintaxe é composta por três partes: a inicialização, a condição e o incremento, conforme mostrado abaixo: 

// Sintaxe básica
for (inicialização; condição; incremento) {
    // bloco de código a ser repetido
}

A inicialização define a variável de controle; a condição é verificada antes de cada repetição; e o incremento altera o valor da variável a cada ciclo. O laço continua enquanto a condição for verdadeira.

Exemplo:

Abaixo está um exemplo prático de uso do for para imprimir os números de 1 a 5 na tela: 

// Exibe números de 1 a 5
#include <stdio.h>

int main() {
    int i;

    for (i = 1; i <= 5; i++) {
        printf("Número: %d\n", i);
    }

    return 0;
}

Neste exemplo, a variável i é inicializada com 1. O laço continua enquanto i <= 5. A cada iteração, i é incrementada em 1 e o valor é exibido com printf. Quando i atinge 6, a condição se torna falsa e o laço é encerrado.
