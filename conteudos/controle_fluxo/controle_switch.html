Estrutura de Controle: switch, case

A estrutura de controle switch em C é usada para selecionar um entre vários blocos de código a serem executados, com base no valor de uma expressão. Ela é especialmente útil quando se tem múltiplas condições discretas para verificar, tornando o código mais legível e organizado em comparação com múltiplos if-else. A sintaxe básica do switch é a seguinte: 

// Exemplo básico com switch
switch (expressão) {
    // bloco de código a ser executado com base no valor da expressão
    case valor1:
        // código a ser executado se expressão == valor1
        break;
    case valor2:
        // código a ser executado se expressão == valor2
        break;
    default:
        // código a ser executado se nenhum dos casos anteriores for atendido
}

A expressão dentro dos parênteses é avaliada e comparada com os valores especificados em cada case. Se houver uma correspondência, o bloco de código associado a esse case será executado até que um comando break seja encontrado, o que encerra a execução do switch. O comando break é importante para evitar a execução "em cascata" dos casos subsequentes.

O default é opcional e será executado se nenhum dos casos anteriores corresponder ao valor da expressão. Ele funciona como um "else" em uma estrutura if-else.

A figura a seguir ilustra um algoritmo que verifica se o a variável valor é igual a 0, 1 ou 2.

Abaixo, veja um exemplo prático completo, que solicita ao usuário um número entre 1 e 3 e exibe uma mensagem correspondente ao número informado. 

// Exemplo completo com switch
#include <stdio.h>

int main() {
    int numero;

    printf("Digite um número entre 1 e 3: ");
    scanf("%d", &numero);

    switch (numero) {
        case 1:
            printf("Você digitou um.\n");
            break;
        case 2:
            printf("Você digitou dois.\n");
            break;
        case 3:
            printf("Você digitou três.\n");
            break;
        default:
            printf("Número inválido! Por favor, digite entre 1 e 3.\n");
    }

    return 0;
}

Neste exemplo, o programa solicita ao usuário que digite um número entre 1 e 3. Com base no número informado, o switch executa o bloco de código correspondente ao case adequado, exibindo uma mensagem. Se o número não estiver entre 1 e 3, o bloco do default é executado, informando que o número é inválido.
