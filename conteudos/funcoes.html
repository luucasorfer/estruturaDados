Funções

Funções em C são blocos de código que executam tarefas específicas. Elas permitem reutilizar código em diferentes partes do programa, tornando o desenvolvimento mais organizado, legível e fácil de manter.

Definição

Para definir uma função em C, usamos a seguinte estrutura: 

tipo_retorno nome_da_funcao(tipo parametro1, tipo parametro2, ...) {
    // corpo da função
    return valor;
}

Aqui:

  • tipo_retorno: indica o tipo de dado que a função vai devolver. Pode ser, por exemplo, int, float, ou void caso não retorne nada.
  • nome_da_funcao: o nome que identifica a função.
  • parametro1, parametro2, ...: são os valores que a função recebe para trabalhar.

Exemplo prático:

int somar(int a, int b) {
    return a + b;
}

Chamada da Função

Para usar uma função, basta chamá-la pelo nome, passando os argumentos necessários entre parênteses: 

int resultado = somar(3, 5);

Aqui, chamamos a função somar com os valores 3 e 5, e guardamos o resultado na variável resultado.

Parâmetros

Parâmetros são variáveis que recebem os valores enviados para a função. Eles podem ter qualquer tipo válido em C, como int, float, ou até arrays.

Veja um exemplo de função com parâmetro:

void exibir_mensagem(char nome[]) {
    printf("Olá, %s!\n", nome);
}

E para chamar essa função:

exibir_mensagem("Maria");

Retorno

Funções em C podem devolver um valor usando a palavra-chave return. O tipo do valor retornado precisa ser compatível com o tipo declarado na definição da função.

Exemplo de função que retorna um valor:

float calcular_area(float raio) {
    return 3.14 * raio * raio;
}

Usando o valor retornado:

float area = calcular_area(5.0);
