Introdução à Linguagem C

Uma linguagem de programação é um método padronizado para comunicar instruções para um computador. É um conjunto de regras sintáticas e semânticas usadas para definir um programa de computador. Permite que um programador especifique precisamente sobre quais dados um computador vai atuar, como estes dados serão armazenados ou transmitidos e quais ações devem ser tomadas sob várias circunstâncias.

Esta é uma introdução à linguagem C, é uma linguagem de programação de propósito geral, estruturada e imperativa. Foi desenvolvida em 1972 por Dennis Ritchie nos laboratórios da AT&T Bell, a linguagem C se tornou uma das linguagens mais influentes na história da computação.

C é considerada uma linguagem de "médio nível", combinando elementos de linguagens de alto nível com a funcionalidade de linguagem de montagem.

Sintaxe e semântica na programação

A sintaxe de uma linguagem de programação é o conjunto de regras que define a estrutura correta dos programas escritos nessa linguagem. Inclui a forma como os comandos, declarações, expressões e outros elementos devem ser organizados para que o código seja compreendido pelo compilador ou interpretador.

A semântica, por outro lado, refere-se ao significado dos elementos da linguagem. Enquanto a sintaxe se preocupa com a forma, a semântica se preocupa com o conteúdo e o comportamento do código. Ela define como as construções sintáticas se traduzem em ações ou resultados durante a execução do programa.

Em resumo, a sintaxe é sobre como escrever o código corretamente, enquanto a semântica é sobre o que o código realmente faz quando é executado. O compilador auxilia a verificar se a sintaxe e a semântica de um programa está correta. Durante os tópicos, serão apresentadas as estruturas básicas da linguagem.

Primeiro Programa em C

Todo programa em C começa com a função main(). Vamos começar com um programa simples que imprime "Hello, World!" na tela. Este é um exemplo clássico usado para ilustrar a estrutura básica de um programa em C. 

#include <stdio.h>

  int main() {
    printf("Hello, World!\n");
    return 0;
  }

Vantagens

  • Alto desempenho
  • Controle de baixo nível
  • Portabilidade
  • Grande base de código existente

Desafios

  • Gerenciamento manual de memória
  • Ausência de orientação a objetos
  • Verificações em tempo de execução limitadas
  • Curva de aprendizado mais íngreme
