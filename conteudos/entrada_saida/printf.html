Escreva: printf

Em determinadas situações precisamos mostrar ao usuário do programa alguma informação. Para isso, existe um comando na programação que exibe dados ao usuário. No portugol a instrução de saída de dados para a tela é chamada de "escreva", pois segue a ideia de que o algoritmo está escrevendo dados na tela do computador.

A função printf() é utilizado quando deseja-se escrever/mostrar informações no console da IDE, ou seja, é um comando de saída de dados.Ela pertence à biblioteca <stdio.h> e permite imprimir valores de variáveis, textos fixos e resultados de expressões.

Para utiliza-lo você deverá escrever este comando e entre parênteses colocar a(s) variável(eis) ou texto que você quer mostrar no console. Lembrando que quando você utilizar textos, o texto deve estar entre aspas. A sintaxe para utilização deste comando está demonstrada a seguir: 

// Sintaxe básica
printf("texto ou formato", variáveis);
    
// Exemplo:
int idade = 25;
printf("Idade: %d\n", idade);

A string de formato pode conter texto comum e especificadores de formato que serão substituídos pelos valores das variáveis.

Especificadores de Formato

Formato Tipo de dado Exemplo
%d int (inteiro) printf("Valor: %d", 10);
%f float ou double printf("PI: %.2f", 3.14);
%c char (caractere) printf("Letra: %c", 'A');
%s string (cadeia de caracteres) printf("Nome: %s", "Ana");
%% Imprime o caractere '%' literal printf("Taxa: 100%%");

Ao usar printf(), o número de especificadores de formato deve corresponder à quantidade de argumentos passados, ou ocorrerá um comportamento inesperado.

Para controlar o número de casas decimais, use formatos como %.2f para mostrar dois dígitos após a vírgula.
