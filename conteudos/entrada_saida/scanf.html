Leia: scanf

Em programação, muitas vezes precisamos obter dados do usuário para que o programa possa processá-los. Para isso, existe um comando que permite ler informações fornecidas pelo usuário durante a execução do programa. No portugol, a instrução de entrada de dados é chamada de "leia", pois o algoritmo está lendo dados fornecidos pelo usuário.

A função scanf() é utilizada em C para ler dados do teclado e armazená-los em variáveis. Ela também faz parte da biblioteca <stdio.h> e é fundamental para obter entrada do usuário durante a execução do programa.

Para utilizá-la, você deve escrever o comando seguido de parênteses, onde dentro dos parênteses você especifica o formato dos dados que deseja ler e as variáveis onde esses dados serão armazenados. A sintaxe básica para usar o scanf() é a seguinte: 

// Sintaxe básica
scanf("formato", &variável);

// Exemplo:
int idade;
scanf("%d", &idade);

Note que usamos o símbolo & (e comercial) para passar o endereço da variável. Isso é necessário para que o scanf() possa armazenar o valor digitado diretamente nela.

Especificadores de Formato

Formato Tipo de dado Exemplo
%d int (inteiro) scanf("%d", &idade);
%f float scanf("%f", &altura);
%lf double scanf("%lf", &peso);
%c char scanf(" %c", &letra);
%s string (sem espaços) scanf("%s", nome);

Quando for ler um caractere com %c, use um espaço antes do especificador: " %c". Isso evita que o scanf() leia um caractere invisível (como Enter) que ficou no buffer.

O %s lê uma palavra até encontrar um espaço. Para ler frases completas com espaços, é necessário utilizar outras técnicas como fgets().

Exemplo:

// Programa que lê nome e idade
#include <stdio.h>

int main() {
    char nome[50];
    int idade;

    printf("Digite seu nome: ");
    scanf("%s", nome);

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("\nOlá, %s! Você tem %d anos.\n", nome, idade);

    return 0;
}
