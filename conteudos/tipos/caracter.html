Tipo Caracter (char) em C

Em C, o tipo char é usado para representar um único caracter — seja uma letra, número, símbolo ou espaço. Por exemplo, para armazenar 'X' ou '0' em um jogo da velha, ou apenas ler um caractere qualquer do usuário.

Exemplo com char

#include <stdio.h>

int main() {
  char c = 'A';

  printf("Você digitou: %c\n", c);

  printf("Digite um caractere: ");
  scanf(" %c", &c);

  printf("O caractere lido foi: %c\n", c);

  return 0;
}
Item Representação em C
Palavra-chave char
Declaração simples char c;
Declaração com valor char letra = 'Z';
Leitura (entrada) scanf(" %c", &c); (note o espaço antes do %c para ignorar espaços em branco)
Escrita (saída) printf("%c", c);
Tamanho típico 1 byte*
Faixa de valores Depende da codificação (ASCII, UTF‑8, etc.)

O tipo char ocupa geralmente 1 byte, mas a faixa de caracteres depende da codificação do sistema (como ASCII ou UTF‑8). Use sizeof(char) para confirmar o tamanho no seu compilador.
