Tipo void em C

Em C, o tipo void é usado para indicar ausência de tipo. Ele serve tanto para funções que não retornam valor quanto para indicar ponteiros genéricos.

Função que não retorna valor

#include <stdio.h>

void saudacao(void) {
  printf("Olá, mundo! (sem retorno)\n");
}

int main() {
  saudacao();
  return 0;
}

Ponteiro genérico void*

#include <stdio.h>

int main() {
  int valor = 42;
  void* ptr = &valor;

  printf("Endereço de memória: %p\n", ptr);
  
  // É necessário fazer casting para desreferenciar
  int* intPtr = (int*) ptr;
  printf("Valor apontado: %d\n", *intPtr);

  return 0;
}
Uso Exemplo em C
Função sem retorno void func(void) { ... }, chamada via func();
Ponteiro genérico void *ptr;, conversão (cast) necessária para uso

Lembrete: ao usar void*, é obrigatório fazer cast antes de acessar o valor apontado, para garantir tipagem correta e evitar comportamento indefinido.
