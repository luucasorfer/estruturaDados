Tipo Inteiro (int) em C

O tipo int em C é utilizado para armazenar números inteiros, ou seja, sem ponto flutuante. Ele é um dos tipos mais utilizados na linguagem.

Item Representação em C
Palavra-chave int
Declaração simples int x;
Declaração com valor int idade = 25;
Leitura (entrada) scanf("%d", &x);
Escrita (saída) printf("%d", x);
Tamanho típico 4 bytes (pode variar)
Faixa de valores −2.147.483.648 a +2.147.483.647
Valor padrão Indefinido (lixo de memória)

Exemplo de uso

#include <stdio.h>

int main() {
  int numero;

  printf("Digite um número inteiro: ");
  scanf("%d", &numero);

  printf("Você digitou: %d\n", numero);

  return 0;
  }

O tamanho do tipo int pode variar entre sistemas (por exemplo, 2 ou 4 bytes). Use sizeof(int) para verificar o tamanho no seu compilador.
