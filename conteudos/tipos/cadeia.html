Tipo char[] em C

Em C, não existe um tipo específico para representar cadeias de caracteres (strings). Em vez disso, utilizamos um vetor de caracteres (char[]), que é uma sequência de caracteres terminada por um caractere nulo ('\0'). Essa abordagem permite armazenar e manipular textos de forma eficiente.

Exemplo com char[] 

#include <stdio.h>

int main() {
  char nome[50];

  printf("Digite seu nome: ");
  scanf("%49[^\n]", nome);

  printf("Olá, %s!\n", nome);

  return 0;
}
Item Representação em C
Declaração simples char nome[50];
Inicialização com valor char saudacao[] = "Olá, Mundo!";
Leitura (entrada) scanf("%49[^\n]", nome);
Escrita (saída) printf("%s", nome);
Formato de escrita e leitura %s (string)
Tamanho típico Depende do número de caracteres alocados
Terminação Sempre termina com '\0'

Atenção ao uso de %s

O especificador %s é usado para ler e imprimir strings com scanf() e printf(). No entanto, scanf("%s", ...) para na primeira ocorrência de espaço. Para ler nomes compostos, utilize %[^\n] como no exemplo acima.

Manipulação com <string.h>

Para manipular strings em C, você pode usar a biblioteca <string.h>, que oferece funções úteis como strcpy, strlen, strcmp, entre outras. 

#include <stdio.h>
#include <string.h>

int main() {
  char texto1[20] = "Olá";
  char texto2[20];
  
  strcpy(texto2, texto1); // Copia texto1 para texto2
  printf("Texto2: %s\n", texto2);

  int tamanho = strlen(texto2);
  printf("Tamanho de texto2: %d\n", tamanho);

  return 0;
}

Sempre reserve espaço suficiente no vetor para todos os caracteres desejados +1 para o caractere nulo '\0'.
