Tipo bool em C

Em C, o tipo lógico é representado por bool (disponível via <stdbool.h>), que pode assumir apenas dois valores: true ou false.

Exemplo com bool

#include <stdio.h>
#include <stdbool.h>

int main() {
  bool ligado = true;
  bool conectado = false;

  printf("Ligado: %s\n", ligado ? "true" : "false");
  printf("Conectado: %s\n", conectado ? "true" : "false");

  return 0;
}
Item Representação em C
Palavra-chave bool (true/false)
Include necessário #include <stdbool.h>
Valor verdadeiro true
Valor falso false
Formato para saída Use operador ternário para converter para string

Operadores Lógicos Comuns

#include <stdio.h>
#include <stdbool.h>

int main() {
  bool a = true;
  bool b = false;

  // Operador E
  bool e = a && b;
  // Operador OU
  bool ou = a || b;
  // Negação
  bool nao_a = !a;

  printf("a && b = %s\n", e ? "true" : "false");
  printf("a || b = %s\n", ou ? "true" : "false");
  printf("!a = %s\n", nao_a ? "true" : "false");

  return 0;
}

Lembre-se: em C, bool é representado como inteiro internamente (0 para falso, 1 para verdadeiro), e documentos como o compilador podem tratar qualquer valor não zero como true.

Exemplo com Expressão Relacional 

#include <stdio.h>
#include <stdbool.h>

int main() {
  int x = 10, y = 20;
  bool resultado = x < y && y > 5;

  printf("x < y && y > 5: %s\n", resultado ? "true" : "false");

  return 0;
}
