Tipo Real (float e double) em C

Em situações onde precisamos representar números com casas decimais (ou seja, não inteiros), usamos o tipo float ou double em C. Ele permite armazenar valores de ponto flutuante, como o valor de PI (3.1415926535) ou resultados de divisões não exatas, como 8.0 / 3.0.

Para representar números com casas decimais(pronto flutuante), você pode utilizar:

  • float: precisão simples (aprox. 6 dígitos decimais).
  • double: precisão dupla (aprox. 15 dígitos decimais), mais preciso e comumente utilizado.

Exemplo com float 

#include <stdio.h>

int main() {
  float numero = 3.14159f;

  printf("Valor (float): %f\n", numero);
  printf("Formatado (2 casas): %.2f\n", numero);

  return 0;
}

Exemplo com double 

#include <stdio.h>

int main() {
  double numero = 3.141592653589793;

  printf("Valor (double): %lf\n", numero);
  printf("Formatado (4 casas): %.4lf\n", numero);

  return 0;
}
Tipo Representação em C Tamanho Típico Faixa Aproximada
float Declaração: float x;
Leitura: scanf("%f", &x);
Escrita: printf("%f", x); 		4 bytes* ±3.4e−38 a ±3.4e+38
double Declaração: double x;
Leitura: scanf("%lf", &x);
Escrita: printf("%lf", x); 		8 bytes* ±1.7e−308 a ±1.7e+308

O tamanho e a precisão de float e double podem variar conforme o sistema e o compilador. Use sizeof(float) ou sizeof(double) para verificar no seu ambiente.

Comparativo: Float vs Double

#include <stdio.h>

void mostrarResultado(float f, double d) {
  printf("Float: %.6f | Double: %.6lf\n", f, d);
}

int main() {
  float f = 1.0f / 3.0f;
  double d = 1.0 / 3.0;

  mostrarResultado(f, d);

  int inteiro = 5;
  float convF = inteiro;
  double convD = inteiro;

  printf("Conversão int → float: %.2f\n", convF);
  printf("Conversão int → double: %.2lf\n", convD);

  return 0;
}

No comparativo acima:

  • O float mostra menos casas decimais (≈6), enquanto o double mostra mais (≈15), demonstrando maior precisão.
  • A conversão de um int para float ou double é automática, mas a precisão depende do tipo destino.
